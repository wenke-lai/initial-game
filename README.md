# Initial Game

An initial game project.

## Requriements

- Python 3.10+
- Poetry 1.8+
- Pygame 2.6+

## Quick Start

1. Clone this project
2. Install all dependencies with poetry `poetry install`
3. Start the game with `poetry run python main.py`
